# Mundial 2026 — Tracker en español

Sitio estático móvil tipo iPhone para seguir el Mundial FIFA 2026:

- Grupos y partidos con marcadores editables.
- Clasificación automática de grupos.
- Mejores terceros y cuadro de eliminación directa.
- Seguimiento por selección.
- Horarios con zonas: Sídney, Bogotá, CDMX y Este de EE. UU.
- Guardado automático en el navegador.

Publicado con GitHub Pages desde este repositorio.

Inspirado en el tracker original de Fernando Mora: https://fdelpiero.github.io/fermora-WC-2026-tracker/
